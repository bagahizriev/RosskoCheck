# 🎨 Генерация иконок для PWA

## Как использовать

1. **Подготовьте SVG иконку:**
    - Размер: 512x512 пикселей
    - Формат: SVG
    - Простой дизайн без мелких деталей
    - Хорошая контрастность

2. **Поместите файл:**

    ```
    public/icon.svg
    ```

3. **Установите зависимости:**

    ```bash
    npm install
    ```

4. **Запустите генерацию:**
    ```bash
    npm run generate-icons
    ```

## Что будет создано

### PNG иконки:

- `icon-16.png` - 16x16px (браузер)
- `icon-32.png` - 32x32px (браузер)
- `icon-48.png` - 48x48px (PWA)
- `icon-72.png` - 72x72px (PWA)
- `icon-96.png` - 96x96px (PWA)
- `icon-144.png` - 144x144px (PWA)
- `icon-192.png` - 192x192px (PWA)
- `icon-512.png` - 512x512px (PWA)

### Apple Touch иконки:

- `apple-touch-icon-120x120.png` - iPhone
- `apple-touch-icon-152x152.png` - iPad
- `apple-touch-icon-180x180.png` - iPhone Plus

### Favicon:

- `favicon-32.png` - для конвертации в ICO

## Создание favicon.ico

После генерации PNG файлов:

1. Откройте https://convertio.co/png-ico/
2. Загрузите `favicon-32.png`
3. Конвертируйте в `favicon.ico`
4. Замените файл `public/favicon.ico`

## Рекомендации по дизайну

- ✅ Простые формы и линии
- ✅ Высокий контраст
- ✅ Читаемость в маленьких размерах
- ✅ Отступы от краев (10-15%)
- ❌ Мелкие детали
- ❌ Тонкие линии
- ❌ Сложные градиенты

## Проверка результата

После генерации иконок:

1. Запустите `npm run build`
2. Проверьте PWA в браузере
3. Добавьте на домашний экран iOS/Android
4. Убедитесь, что иконки отображаются корректно
